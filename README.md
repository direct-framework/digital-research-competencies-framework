# GitHub Pages Redirect

GitHub Pages originally hosted the website for the project
called **RSE Competencies Toolkit**, available at:

https://rsetoolkit.github.io/rse-competencies-toolkit/

The project was later renamed and moved to a new repository:

https://github.com/direct-framework/digital-research-competencies-framework

The Github Pages for this repo is:

https://direct-framework.github.io/digital-research-competencies-framework/

However, the website architecture and design has also changed. The live site is now hosted at:

https://directframework.com/

To avoid breaking old links, this branch now contains only a redirect page (`index.html`) that forwards users to the new official website.
